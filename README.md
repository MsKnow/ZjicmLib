# 浙传移动图书馆
#### 简介
专为浙传图书馆适配的移动app 该有的功能都有了 包括浏览公告 搜索排行 图书搜索 图书详情 登录续借等。内网即可访问。
该版为重构版本 会持续更新。如果觉得好用请分享给身边的人，非常感谢！\\(//∇//)/ 扫码或这里下载 http://fir.im/ZCLib
